# Luther's Small Catechism Translation Tool

A web-based tool developed by Concordia Publishing House to facilitate translations of Luther's Small Catechism into additional languages.

## Access the Tool

**Live Tool:** https://roegnejj.github.io/catechism-translation-tool/

**Test Mode:** https://roegnejj.github.io/catechism-translation-tool/?test=true

## Features

- Translate the complete Small Catechism section-by-section
- Automatic progress saving and tracking
- Preserves markdown formatting
- Works offline after initial load
- Download completed translations as .md files
