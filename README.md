# Luther's Small Catechism Translation Tool

This tool streamlines the translation process by providing the English version of Luther's Small Catechism in an easy-to-use interface. Translators work section-by-section, with automatic formatting preservation and progress tracking. The tool works offline after initial load and autosaves progress as you work.

## Access the Tool

https://roegnejj.github.io/catechism-translation-tool/

## What's Included:

The complete Small Catechism (all 11 sections)
App interface terms for our mobile application
Apple AppStore & Google Play store listing content

## Features

- Translate the complete Small Catechism section-by-section
- Automatic progress saving and tracking
- Preserves markdown formatting
- Works offline after initial load or download offline text (.txt) files, if preferred
- Download completed translations as .md files

## Getting Started

The tool includes a built-in Quick Start Guide that walks you through the translation process. Completed translations can be emailed to jeremy.roegner@cph.org.

You can preview the what the final product will look like at catechism.cph.org to see how translations appear in our mobile application.
